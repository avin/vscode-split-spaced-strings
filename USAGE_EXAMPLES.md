# Примеры использования Auto-Collapse функции

## Базовое использование

### 1. Включение авто-схлопывания

Откройте настройки VS Code:
- `Ctrl+,` (Windows/Linux) или `Cmd+,` (Mac)
- Найдите "Split Spaced Strings"
- Включите "Auto Collapse On Save"

Или добавьте в `settings.json`:
```json
{
  "splitSpacedStrings.autoCollapseOnSave": true
}
```

### 2. Работа с развернутыми строками

#### Разворачивание строки
1. Поместите курсор внутрь строки
2. Нажмите `Alt+Shift+S`
3. Строка развернется, и появится желтая подсветка (если включено auto-collapse)

#### Пример:
```tsx
// До разворачивания
<div className="flex items-center justify-between p-4 bg-white">

// После Alt+Shift+S (с включенным auto-collapse - будет подсвечена желтым)
<div className="
  flex
  items-center
  justify-between
  p-4
  bg-white
">
```

### 3. Автоматическое схлопывание при сохранении

Когда вы сохраняете файл (`Ctrl+S`), все подсвеченные строки автоматически вернутся в однострочный формат:

```tsx
// После сохранения
<div className="flex items-center justify-between p-4 bg-white">
```

## Сценарии использования

### Сценарий 1: Временное редактирование длинных строк

1. У вас есть длинная строка с классами Tailwind
2. Нужно добавить/удалить несколько классов
3. Разворачиваете строку для удобства редактирования
4. Редактируете нужные классы
5. Сохраняете файл - строка автоматически схлопывается

### Сценарий 2: Работа с несколькими строками

Можно развернуть несколько строк в файле:
```tsx
// Файл с несколькими развернутыми строками
<div className="
  flex
  items-center
">
  <span className="
    text-lg
    font-bold
    text-gray-900
  ">
    Hello
  </span>
</div>
```

При сохранении все помеченные строки схлопнутся:
```tsx
<div className="flex items-center">
  <span className="text-lg font-bold text-gray-900">
    Hello
  </span>
</div>
```

### Сценарий 3: Постоянное разворачивание (без auto-collapse)

Если вы хотите оставить строки развернутыми:
1. Выключите "Auto Collapse On Save"
2. Разворачивайте строки как обычно
3. Они останутся развернутыми после сохранения
4. Схлопнуть можно вручную через `Alt+Shift+S`

## Визуальные индикаторы

### Подсветка отслеживаемых строк
- **Желтый фон**: Строка будет схлопнута при сохранении
- **Полоска в overview ruler**: Показывает позицию отслеживаемых строк в файле
- **Hover сообщение**: Наведите курсор на подсвеченную строку для информации

## Отключение функции

Чтобы отключить автоматическое схлопывание:
```json
{
  "splitSpacedStrings.autoCollapseOnSave": false  // Значение по умолчанию
}
```

При отключенной настройке:
- Строки не будут подсвечиваться
- Схлопывание происходит только вручную через `Alt+Shift+S`
- Поведение как в оригинальной версии расширения
